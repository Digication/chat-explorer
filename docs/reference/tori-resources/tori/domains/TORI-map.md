# TORI: Taxonomy of Reflective Inquiry — Skill Map

## How to Use This Map

This map is the entry point to the TORI (Taxonomy of Reflective Inquiry) skill system. Each category below links to a dedicated skill file containing the full definition, reflection model instructions, grade-level examples, related fields, and theoretical foundations.

**IMPORTANT — Multi-Category Selection Required:**
When using TORI to support a user's reflective inquiry, do NOT simply choose the single best-matching category. A user's need will often span multiple categories across different domains. You MUST scan ALL domains and categories below, then select and read ALL relevant skill files that apply. Reflection is multidimensional — a single experience may involve cognitive, emotional, social, and personal growth dimensions simultaneously. Err on the side of including more categories rather than fewer.
